A beautiful, interactive web application designed to capture your daily thoughts, feelings, and memories. Built from scratch with a stunning modern Glassmorphism aesthetic and smooth micro-animations.

## ✨ Features
* **Glassmorphism UI:** Gorgeous, semi-transparent frosted glass cards blending perfectly into a slow-moving, animated gradient background.
* **Mood Tracker:** Easily record how you felt on any given day by selecting an interactive emoji.
* **Persistent Storage:** Never lose a memory! The app utilizes browser `LocalStorage` to ensure your entries stay saved even if you close the tab or refresh the page.
* **Dynamic Animations:** Fluid hover effects, stagger-fade load animations, and completely responsive interactions without any page reloads.
* **Clean Typography:** Built using Google's sleek `Inter` font matrix and FontAwesome icons.

## 🚀 How to Run Locally 
It's incredibly simple to use! Because it is constructed using Vanilla web languages, you don't need any complex servers or installations. 

1. Download or Clone the repository:
   ```bash
   git clone https://github.com/Vishwakarthick1789/PocketDiary.git
   ```
2. Navigate into the folder and simply double-click the `index.html` file to instantly open it entirely in your browser

## 🛠️ Technology Stack
* **Structure:** Semantic HTML5
* **Styling:** Custom CSS3 (CSS Variables, Flexbox, Keyframe Animations, Backdrop-filters)
* **Logic:** Vanilla JavaScript (ES6, DOM Manipulation)

## 📸 Usage
1. Type a Title for your day.
2. Select the emoji that matches your mood.
3. Write your thoughts in the text area.
4. Click **Save Entry** and watch it animate directly onto your timeline below
5. Change your mind? Click the little Trash Can icon to instantly delete a specific entry.
